Travel Record for macOS ✈️🚄
Travel Record 是一款专为 macOS 设计的个人旅行足迹管理工具。它利用 Apple 最新的 SwiftData 技术，帮助你记录每一次飞行和铁路旅程，并能智能地根据时间自动归档行程。

🌟 核心功能
智能行程分类：

  即将到来 (Upcoming)：实时监控时间，一旦航班或列车起飞/发出，自动将其移入历史记录。

  历史足迹 (History)：按照时间倒序排列你所有的旅行回忆。

深耕铁路系统：

  特别针对不同国家的铁路逻辑进行优化（如中国铁路的席别：商务座、一等座、二等卧等）。

  支持运营商与席位的级联选择。

原生 macOS 体验：

  采用 SwiftUI 开发，完美契合 macOS 系统的视觉风格。

  支持标准的 macOS 交互，如表单输入、日期选择器等。

数据安全：

  所有数据均通过 SwiftData 存储在本地，保护隐私，无需联网即可查看。

🛠️ 技术实现
框架: SwiftUI (macOS 14.0+)

持久化: SwiftData

自动刷新: 使用 Combine 定时器驱动视图，确保在应用开启状态下行程分类始终准确。

数字格式化: 针对航班号优化，去除了千分位逗号干扰，更符合行业习惯。

🚀 安装与运行
环境要求

macOS: 14.0 或更高版本

Xcode: 15.0 或更高版本
